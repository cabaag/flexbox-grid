Flex ordering
Add the class .flex-order to reorder elements within layout container. The values goes from -20 to 20.
.flex-order--1
.flex-order-4
.flex-order-1
.flex-order--3

  <div class='flex flex-order--1'>
    <div class='box'>.flex-order--1</div>
  </div>
  <div class='flex flex-order-4'>
    <div class='box'>.flex-order-4</div>
  </div>
  <div class='flex flex-order-1'>
    <div class='box'>.flex-order-1</div>
  </div>

  </div>

Flex offsets
Add the class .flex-offset to a layout's child element to set its offset percentage within the layout container. Value must be multiples of 5, 33 or 66.
.flex.flex-offset-25
.flex-40.flex-order--1
<div class='layout-row layout-padding'>
  <div class='flex flex-offset-25'>
    <div class='box'>.flex.flex-offset-25</div>
  </div>
  <div class='flex-40 flex-order--1'>
    <div class='box'>.flex-40.flex-order--1</div>
  </div>
</div>

Layout alignment
The .layout-align class is composite for two words, the first indicates the horizontal alingment and the second indicates vertical alignment. If one word is passed it'kk works but only for horizontal alignment.
<div class="layout-row layout-wrap layout-padding">
  <div class="flex-40 layout-column layout-align-center">
    <div class="flex layout-row layout-align-space-around-center">
      <div class="box"></div>
      <div class="box"></div>
      <div class="box"></div>
    </div>
    <div class="flex box"></div>
    <div class="flex box"></div>
  </div>
  <div class="flex-20 layout-column layout-align-space-around-stretch">
    <div class="flex-30 box"></div>
    <div class="flex-30 box"></div>

</div>

Fluid

Converts the container into a full width container, just replace .container for .container-fluid

<div class='container-fluid'>
  <div class='layout-row'>
    <div class='flex-xs-100 flex-sm-50 flex-md-60 flex-80'>
      <div class='box'>Container fluid</div>
    </div>
  </div>
</div>
